# TodayMenu

## Purpose
> <b>Today Menu?</b><br>
> 오늘 식사 메뉴를 고를 때 어려움이 많다. 이럴 때 메뉴를 추천해주고 그에 따른 동네 맛집까지 추천해주고 카카오톡으로 메세지를 받아볼 수 있다.<br>
> 또한 날씨에 따른 메뉴를 추천 받고 싶을 때 오늘 날씨 메뉴 추천을 받을 수 있는 Web Page이다.<br>
